Felder markieren für Sprungfolge:

|         | V  vorne |          |
| ------- | -------- | -------- |
| L Links | 0        | R Rechts |
|         | H hinten |          |

* 00 - VV/HK - 00 - RR -00 - HH/HK - 00 - LL 
* 00 - VV/HK - 00 - VV/HK
* 00 - LL - 00 - HH/HK - 00 - RR - VV/HK
* 00 - RL - 00 - RL - 00
* 00 - 34KK - 00 - 34KK
* VV
* 00 - RL/KK - 00 - RL/KK
* RR/BK - 00 - LL/BK
* 00 - RL/KK - 00 - RL/KK
* VV - End

00 - RL/KK ist ein Hampelmann
