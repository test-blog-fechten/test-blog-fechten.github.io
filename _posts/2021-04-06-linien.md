#### Blöße (freie Trefffläche) 

* Ein Kreuz vor dem Florettfechter teilt die vier Blößen ein.
  * Oben außen
  * Oben innen
  * Unten innen
  * Unten außen 

Allerdings teilt die Klinge/Glocke des Gegners die Blößen ein. (nicht sein Körper) 

Außerdem wechselt die Fläche mit der Bewegung der Klinge/Glocke.

Beispiel: Indirekte Attacke (mit Umgehung) / obwohl die Klinge sich nicht zur Trefffläche verschiebt. Spricht man davon das die Attacke auf eine andere Blöße trifft.   

* Grafiken z.B. S.70  Ich lerne fechten können leicht falsch verstanden werden. 
* Gespiegelt bei Links und Rechts-Händern.



-----

Manche teilen das auch in Treffflächen (nach den Paraden) ein. ( z.B. Florett )

Die Treffflächen beim Gegner Linkshänder

|  Quart(4)  | Sixed(6) | <-- Waffen Hand |
| :--------: | :------: | :-------------: |
| Septime(7) | Octav(8) |                 |

oder Rechtshänder

| Waffen Hand --> | Sixed(6) |  Quart(4)  |
| :-------------: | :------: | :--------: |
|                 | Octav(8) | Septime(7) |
